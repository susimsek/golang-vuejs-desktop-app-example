## Prerequisites

* Nodejs v12.x
* Npm 6.x
* Golang 1.16.x
* Wails 1.16.x

## Installation 

```sh
cd cpustats
```

```sh
wails build
```

```sh
./build/cpustats
```

## Packaging

```sh
cd cpustats
```

```sh
wails build -p
```
